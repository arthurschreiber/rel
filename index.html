Rel is a SQL AST manager for Node JS

It is a straight port of https://github.com/rails/arel. Although it does have some changes. For me details checkout the Github page.

Simple Usage


users = new Rel.Table 'users'
users.project Rel.star()
users.toSql()

Produces

 SELECT * FROM users

Fancy Usage

users.where(users.column('name').eq('amy')).project(users.column('id'))

Generates

SELECT users.id FROM users WHERE users.name = 'amy'

Installation

npm install rel

For more examples and details look at the Github page.
